## Como publicar no GitHub Pages
1. Crie um repositório público no GitHub chamado `portfolio` (ou o nome que preferir).
2. Faça upload dos arquivos (`index.html`, pasta `css/`).
3. No GitHub, vá em **Settings → Pages**.
4. Em **Source**, escolha a branch `main` (ou `master`) e a pasta `/ (root)`.
5. Clique em **Save**. O site ficará disponível em `https://<seu-usuario>.github.io/<nome-do-repositorio>/`.
